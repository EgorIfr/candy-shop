## Шаг 2: Запуск сервера

```bash
npm start
```

## Шаг 3: Вход в админ-панель

1. Откройте приложение в браузере
2. Перейдите на страницу входа: `/login`
3. Введите:
   - **Логин:** `admin`
   - **Пароль:** `admin`
4. После входа вы увидите ссылку "Админ" в шапке сайта
5. Перейдите в админ-панель: `/admin`

## Функционал админ-панели

- **Добавление товаров:**
  - Нажмите "+ Добавить товар"
  - Заполните форму (название, описание, цена, категория, изображение)
  - Нажмите "Создать товар"

- **Удаление товаров:**
  - В списке товаров нажмите кнопку "Удалить" на нужном товаре
  - Подтвердите удаление

## Регистрация новых пользователей

Обычные пользователи могут зарегистрироваться на странице `/register`.
Только пользователи с ролью `admin` имеют доступ к админ-панели.
